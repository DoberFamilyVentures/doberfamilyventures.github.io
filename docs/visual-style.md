# Dober Family Ventures — Visual Style Guide

## Overall visual direction

The visual identity should feel established, calm, and durable. It should look more like a long-term family operating company than a modern SaaS landing page.

## Logo direction

Primary mark:

- Serif `D`
- Tree / branch symbol integrated inside the letter
- Should communicate legacy, roots, growth, family, and long-term stewardship

Logo variations needed:

- Full-color logo on dark navy background
- Full-color logo on light background
- White transparent version
- Black transparent version
- Favicon version with dark navy background and white `D` / green tree

## Favicon direction

Use the standalone `D` + tree mark.

Recommended favicon characteristics:

- Square canvas
- Dark navy background
- White `D`
- Green tree symbol
- Slightly rounded corners are acceptable for app-icon style usage
- Avoid tiny tagline or full wordmark in favicon

## Color direction

The brand palette should center around:

- Deep navy: trust, maturity, stability
- Cream / off-white: warmth and readability
- Evergreen: growth, household, family, life
- Muted gold: value, stewardship, long-term quality

Avoid overly bright colors. The site should not feel neon, techy, or trendy.

## Typography

Use a strong serif typeface for major headings. The serif gives the brand a sense of permanence and legacy.

Use a readable sans-serif for body copy, navigation, labels, and smaller UI elements.

Recommended typography behavior:

- Large serif hero heading
- Serif section headings where appropriate
- Uppercase small-label text with letter spacing
- Body copy should be calm and easy to read

## UI style

Cards should feel substantial but not flashy.

Recommended style:

- Dark navy card backgrounds with subtle gradients
- Thin borders with low opacity
- Rounded corners
- Muted shadows
- Gold accent lines used sparingly
- Green labels used as section/category accents

Avoid:

- Bright gradients
- Excessive animation
- Overly glossy SaaS effects
- Loud hover interactions
- Cluttered iconography

## Layout direction

The site should have generous spacing and strong hierarchy.

The hero should feel calm and composed:

- Left side: mission and text
- Right side: portfolio structure / visual explanation

The page should not become overly dense. Dober Family Ventures is a parent brand, so clarity and restraint matter more than volume.
